## 项目结构概述

```bash
├─api                        # 与后端交互的接口函数
├─assets                     # 项目的图片、字体、基础样式等资源文件
├─components                 # 可复用的组件
├─directive                  # 自定义指令
├─mock                       # 开发阶段的请求拦截响应模拟函数
├─router                     # 系统路由
├─store                      # 系统状态管理
├─utils                      # 系统所需工具类
└─views                      # 系统的所有页面
```
